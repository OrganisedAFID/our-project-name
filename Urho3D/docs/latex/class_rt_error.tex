\doxysubsection{Detailed Description}
Exception handling class for \mbox{\hyperlink{class_rt_audio}{Rt\+Audio}} \& Rt\+Midi. 

The \mbox{\hyperlink{class_rt_error}{Rt\+Error}} class is quite simple but it does allow errors to be \char`\"{}caught\char`\"{} by \mbox{\hyperlink{class_rt_error_ab04667aae01bffc354a9ac6bda6903ac}{Rt\+Error\+::\+Type}}. See the \mbox{\hyperlink{class_rt_audio}{Rt\+Audio}} and Rt\+Midi documentation to know which methods can throw an \mbox{\hyperlink{class_rt_error}{Rt\+Error}}.
