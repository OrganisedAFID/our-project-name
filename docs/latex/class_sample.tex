\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sample}{Sample}} class, as framework for all samples.
\begin{DoxyItemize}
\item Initialization of the \mbox{\hyperlink{namespace_urho3_d}{Urho3D}} engine (in Application class)
\item Modify engine parameters for windowed mode and to show the class name as title
\item Create \mbox{\hyperlink{namespace_urho3_d}{Urho3D}} logo at screen
\item Set custom window title and icon
\item Create Console and Debug HUD, and use F1 and F2 key to toggle them
\item Toggle rendering options from the keys 1-\/8
\item Take screenshot with key 9
\item Handle Esc key down to hide Console or exit application
\item Init touch input on mobile platform using screen joysticks (patched for each individual sample) 
\end{DoxyItemize}
